## MERN React Shop

#### Simple shop made with React, Redux, Thunk, Styled Components, Framer Motion and Axios.

#### Backend: MongoDB, Node.js

Front hosted on Netlify, Back on Heroku.

#

Live:
https://objective-jones-10a8d9.netlify.app/

#### Install

    - clone this project
    - cd into the project

    Client
    - cd client
    - npm install
    - npm start

    Server
    - cd server
    - npm install
    - npm run local
